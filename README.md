# Floowe Product Analysis

This repository contains a product analysis and feature proposal exercise based on the Floowe platform.

Floowe is an AI-powered content generation and multi-channel publishing platform designed to help businesses automate content marketing workflows.

The goal of this exercise was to analyse the product, understand its user flows and architecture, and propose improvements aligned with its core value proposition.

# Approach

The analysis followed a structured product discovery approach:

1. Product understanding  
2. User flow identification  
3. Event Storming to map system behaviour  
4. Feature proposal aligned with business value

---

# AI Support

Claude AI was used during the analysis phase to assist with:

- brainstorming potential feature ideas
- structuring domain events
- validating product workflows

All outputs were reviewed and refined to align with the identified product architecture and user flows.

---

# Key Insights

The Floowe platform focuses on three main capabilities:

- AI-powered content generation
- multi-channel content publishing
- SEO-driven traffic growth

The proposed features strengthen these capabilities by introducing:

- analytics for content performance
- scheduling for consistent publishing
- brand voice personalization for AI-generated content
